# 两个小赌博游戏

假设我的客厅里有一个装有90个高尔夫球的水桶，我们将用它玩两轮赌博游戏。 （哇，您在想，那真是个大问题。是的，我们正在为此而发挥想象力。）

您的工作将是预先选择两种赌博的条件（在抽出任何球之前）。

然后我会闭上眼睛，将球彻底打乱，拔出一个球，向您展示，放回去，然后重复。 （您将通过屏幕实时观看我，因此我不会作弊）。 得到它了？ 好的，这是最重要的事情-您对水桶有什么了解？
